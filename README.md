# 📖 Project Overview

This project is a modern AI agency landing page developed as part of the **MorpheLabs Frontend Internship Assessment**.

The objective was to redesign a professional landing page using only **HTML, CSS, and JavaScript**, while maintaining clean code, responsive design, accessibility, smooth animations, and an engaging user experience.

---

# ✨ Features

- Modern & Premium UI
- Fully Responsive Layout
- Smooth Scroll Animations
- Interactive Navigation
- Animated Hero Section
- Services Section
- FAQ Accordion
- Contact Form
- Back-to-Top Button
- SEO-Friendly Structure
- Pure HTML, CSS & JavaScript

---

# 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- SVG Icons
- CSS Animations
